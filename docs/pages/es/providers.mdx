# Proveedores

El revisor soporta cuatro proveedores de LLM. Configurá el tuyo preferido en `.ai-review.yml`:

```yaml
provider: openai   # openai | anthropic | gemini | ollama
model: gpt-4o-mini # nombre del modelo según el proveedor
```

## Comparativa de proveedores

| Proveedor | Secret de GitHub | Modelo por defecto | Consola |
|---|---|---|---|
| OpenAI | `OPENAI_API_KEY` | `gpt-4o-mini` | [platform.openai.com](https://platform.openai.com) |
| Anthropic | `ANTHROPIC_API_KEY` | `claude-sonnet-4-20250514` | [console.anthropic.com](https://console.anthropic.com) |
| Gemini | `GEMINI_API_KEY` | `gemini-1.5-flash` | [aistudio.google.com](https://aistudio.google.com) |
| Ollama | *(sin key — auto-hospedado)* | configurar con `model:` | [ollama.com](https://ollama.com) |

## OpenAI

Proveedor por defecto. Usa `response_format: json_schema` para output estructurado — el modelo está restringido a JSON válido que coincide con el schema de revisión, sin necesidad de post-procesamiento.

```yaml
# .ai-review.yml
provider: openai
model: gpt-4o-mini   # o gpt-4o para mayor calidad
```

```yaml
# Workflow de GitHub Actions
- run: npx -y @giolabsuy/ai-code-reviewer@latest review-pr
  env:
    OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

## Anthropic

Usa instrucciones de formato JSON explícitas agregadas al system prompt.

```yaml
# .ai-review.yml
provider: anthropic
model: claude-sonnet-4-20250514
```

```yaml
# Workflow de GitHub Actions
- run: npx -y @giolabsuy/ai-code-reviewer@latest review-pr
  env:
    ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

```yaml
# .ai-review.yml
```

```yaml
# Workflow de GitHub Actions
```

## Ollama (auto-hospedado)

Ollama corre localmente y no requiere API key, pero el servicio tiene que estar corriendo y ser accesible desde el entorno donde se ejecuta el revisor.

```yaml
# .ai-review.yml
provider: ollama
model: codellama          # o cualquier modelo que hayas descargado
ollamaUrl: http://localhost:11434   # valor por defecto si se omite
```

Para GitHub Actions, Ollama tiene que correr como un servicio en el job o ser accesible vía una URL de red. Para uso local, simplemente iniciá `ollama serve` antes de ejecutar el revisor.
